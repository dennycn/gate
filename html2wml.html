Html2Wml Test Page

This page illustrates how to call Html2Wml from an HTML form. It also shows that most of the options of Html2Wml can be used in CGI mode.

If you uncheck the "Debug" option, the result will be sent as a real WML deck, which is not viewable with standard web browsers.
